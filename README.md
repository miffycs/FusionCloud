FusionCloud 是一套專為追求清晰進出場信號而設計的綜合型策略指標，結合三大技術面工具：EMA 雲帶趨勢辨識、RSI 金叉死叉動能進出場判斷、以及 MACD 柱狀動能變化，同時支援動能破位與三段式平倉靈敏度調整。

#### 🇹🇼 繁體中文 | <a href="README-en.md">🇺🇸 English</a>

## 🔍 核心邏輯說明

1. EMA 雲帶辨趨勢
- 三條 EMA（短、中、長）組成雲帶，用來判斷多空排列。
- 綠色雲帶：多頭排列（短 > 中 > 長）。
- 紅色雲帶：空頭排列（短 < 中 < 長）。

2. RSI + MACD 動能進場訊號
- 多單進場：「RSI金叉 + MACD柱狀 > 0 + 雲帶多頭排列」。
- 空單進場：「RSI死叉 + MACD柱狀 < 0 + 雲帶空頭排列」。
- 額外進場：當 MACD 柱狀加速連續三根同向，且 RW 動能同向時觸發破位進場。

3. 動態平倉機制（靈敏度可調）
- 提供 高 / 一般 / 低 三種靈敏度選擇，對應不同交易風格。
- 平倉邏輯組合：RSI 死叉 / MACD 柱狀反轉 / RWMo 動能翻轉。

4. 爆發動能破位條件
- 當 MACD 柱狀持續放大 + RWMo 同向動能確認，即視為加速突破。

5. 反轉潛力提示
- 若 RSI、MACD、RWMo 三者出現反向訊號，會顯示小箭頭提示，但不會影響主邏輯進出場。


## 🎯 使用教學
- 進場點：蠟燭上顯示「多」或「空」即為進場訊號。
- 平倉點：「平多」或「平空」出現即為出場建議。
- 背景顏色：綠色代表多單持倉中，紅色代表空單持倉中。
- 靈敏度調整：從設定面板中選擇「高 / 中 / 低」平倉靈敏度。
- 設置警報：支援所有訊號類型的警示（進場、平倉、反轉潛力）。


## ✅ 推薦用法
- 適用於加密貨幣、外匯、期貨等資產。
- 表現最佳於 1分鐘～15分鐘週期的短線策略。
- 不會 repaint，適合量化輔助或手動交易搭配。
